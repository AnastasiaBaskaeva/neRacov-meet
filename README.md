# 📅 neRacov-meet

Корпоративное приложение для планирования встреч сотрудников внутри компании.  
Разработано командой **t34M4 n3 R4k0V**.

---

## 📋 О проекте

**neRacov-meet** — это система для организации рабочих встреч: сотрудники могут создавать встречи, приглашать коллег и управлять своим расписанием прямо со смартфона. Никаких лишних инструментов — всё внутри одного приложения.

### Кому подойдёт

- Небольшим и средним компаниям, которым нужен простой внутренний инструмент для координации встреч
- Командам, работающим в офисе или в гибридном формате
- Всем, кто устал от хаоса в расписании и хочет видеть занятость коллег без лишних созвонов

---

## ✨ Функционал

### Аккаунт
- Регистрация и авторизация сотрудника
- Настройка и редактирование личного профиля

### Встречи
- Создание встречи с выбором даты и временного слота (строго по часам: 9:00–10:00, 14:00–15:00 и т.д.)
- Приглашение коллег на встречу
- Ответ на приглашение: **принял** / **не принял**
- Просмотр списка активных входящих приглашений

### Расписание
- Просмотр собственного расписания встреч в трёх режимах:
  - **День** — почасовая сетка
  - **Неделя** — встречи по дням
  - **Месяц** — обзор занятости с маркерами

## 🚀 Как запустить

### 1. Клонировать репозиторий

```bash
git clone https://github.com/AnastasiaBaskaeva/neRacov-meet.git
cd neRacov-meet
```

### 2. Запустить Backend (Docker)

Убедитесь, что Docker установлен и запущен, затем выполните:

```bash
cd backend
docker compose up --build
```

Сервер запустится на порту **8080**.

### 3. Настроить адрес сервера в Android-приложении

Перед сборкой приложения необходимо указать адрес вашего сервера.

Откройте файл:
```
frontend/app/src/main/java/ru/sicampus/bootcamp2026/android/data/source/Network.kt
```

Найдите блок:
```kotlin
object Network {
    const val HOST = "http://10.0.2.2:8080"
}
```

И замените его в зависимости от того, где запускаете приложение:

```kotlin
object Network {
    // Для эмулятора Android Studio:
    const val HOST = "http://10.0.2.2:8080"

    // Для физического устройства — укажите IP-адрес вашего компьютера в локальной сети:
    // const val HOST = "http://192.168.X.X:8080"
}
```

> **Как узнать свой IP:** на Windows — `ipconfig` в терминале, на macOS/Linux — `ifconfig` или `ip a`. Используйте адрес из раздела Wi-Fi / локальной сети.

### 4. Собрать и запустить Android-приложение

Откройте папку `frontend` в **Android Studio**, дождитесь синхронизации Gradle, затем нажмите **Run**.
